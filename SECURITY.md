# Security Policy

NextBuf 仍处于 `v0.x` 开发阶段，尚未发布稳定支持版本。

## 报告漏洞

请通过 GitHub 仓库的 Private vulnerability reporting / Security Advisory 私下报告安全问题：

<https://github.com/Xwordsman/nextbuf/security/advisories/new>

不要在公开 Issue 中提交可利用细节、真实密钥、用户数据或未修复漏洞的完整复现。

报告建议包含：

- 受影响版本或提交。
- 影响范围和前置条件。
- 最小复现步骤。
- 建议修复或缓解方式（如有）。

项目会在确认问题后协调修复与披露时间。正式版本发布前将补充支持版本和响应时限。
